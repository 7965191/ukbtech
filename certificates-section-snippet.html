СЕРТИФІКАТИ ТА ДОКУМЕНТИ

Ми дотримуємося промислових стандартів для забезпечення якості, екологічної безпеки та відповідності вимогам ЄС.

ISO 9001

Система менеджменту якості

Забезпечує контроль процесів від проєктування до доставки.

ISO 14001

Система екологічного менеджменту

Підтверджує екологічну відповідальність у процесах металообробки.

ISO 5817 (B/C)

для зварювання

Стандарт якості зварних швів, релевантний для конструкцій.

ЄСКД / ISO

для креслень та документації

Український та міжнародний стандарт для технічної документації.

CE-маркування

Machinery Directive 2006/42/EC

Відповідність європейським директивам для продукції.

Сертифікат СС3

для проєктів ПОБ та ПВР

Вищий рівень для пожежної безпеки та охорони праці в будівництві/виробництві.

Ліцензії та Дозволи

Ліцензія на експлуатацію небезпечного обладнання

від Держпраці

Для верстатів, пресів та зварювання.

Дозвіл на викиди забруднюючих речовин

від Міндовкілля

Для екологічного контролю процесів (наприклад, термічної обробки). Сумісний з EU REACH.

EORI-номер

для митниці ЄС

Реєстрація як експортер для безперешкодної торгівлі з ЄС (отримання через ДМС України, безкоштовно).
